# Datarize Frontend 과제 전형

## 프로젝트 설정 및 실행 방법

### 필수 요구사항

- Node.js v20.13.1
- Yarn v1.22.22

### 설치 및 실행

```bash
cd apps
yarn install
yarn start-server
yarn start-client
```

### 접속 정보

- 클라이언트: http://localhost:3000
- 서버: http://localhost:4000
